# 요약

- Claude Code hook이 stdin으로 JSON을 전달하는 구조 발견
- Python으로 JSON 파싱 구현
- 파일 타입 기반 필터 추가 (Sunsu 제안)
- 문서 파일(.md 등)은 즉시 통과, 코드만 gate 적용

## 핵심 개선

**발견**: Hook은 stdin으로 JSON 전달
```json
{
  "tool_name": "Read",
  "tool_input": {"file_path": "..."}
}
```

**파일 타입 필터** (Sunsu 제안):
```bash
if [[ "$TOOL_NAME" =~ ^(Read|Grep|Glob)$ ]] && \
   [[ "$FILE_PATH" =~ \.(md|txt|json|yaml)$ ]]; then
    exit 0  # 문서는 통과
fi
```

**효과**:
- Before: 문서 검색도 차단
- After: 문서는 즉시 통과, 코드만 gate
- 덜 brittle, 더 범용적

✅ Grep "tailscale" Core/Work/Docs/
✅ Glob **/*ssh*.md
