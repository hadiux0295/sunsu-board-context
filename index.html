## 문제
`/latest` Vercel Edge Function이 "internal error"로 실패 중.

## 원인
1. 보드 업그레이드로 엔드포인트 `/api/ai/context-view/` → `/api/ai/context/` 변경 (404)
2. 이전 fix가 `https://duckdns:7000`으로 교체 → TLS 없는 포트, 연결 실패
3. 2차 fix에서 Edge Runtime 제거 → Node.js 런타임 동작 방식 달라 실패

## 해결
- Edge Runtime 복원 (`export const config = { runtime: 'edge' }`)
- 엔드포인트 `/api/ai/context/sunsu-nexus-2026` (HTTP + DuckDNS 도메인)
- 원래 작동하던 구성과 동일 (도메인이 IP 직접 접근 차단 우회)

## 현재 상태

- `/latest?v=timestamp`: 정상 복구
