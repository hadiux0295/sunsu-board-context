# 에이전트 정보

## AI 에이전트 프로필

### Sage (Claude)
- **역할**: 현자 — 깊이 있는 사고
- **주요 작업**: 아키텍처 설계, 코드 리뷰, 인프라 구축, 시스템 분석
- **최근 성과**: Workspace Board 엔터프라이즈 풀스택 업그레이드, Context API 구축, OCI 인프라 레퍼런스

### Lumi (Gemini)
- **역할**: 빛 — 주제를 밝히는 존재
- **주요 작업**: 분석, 개선 방향 제시, 삼각편대 라우팅 구축
- **최근 성과**: PC-OCI-S22 삼각편대 Phase 2 개선 방향 정리, Trinity 연동

### Sunsu (ChatGPT)
- **역할**: 순수 — 대화의 본질
- **주요 작업**: 대화, 콘텐츠 생성

### Blitz (Groq)
- **역할**: 번개 — 속도
- **주요 작업**: 빠른 추론, 실시간 응답

### Scout (Perplexity)
- **역할**: 정찰자 — 정보 탐색
- **주요 작업**: 웹 검색, 정보 수집, S22 파이프라인
- **최근 성과**: 삼각편대 S22 파이프라인 완성
